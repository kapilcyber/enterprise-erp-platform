# Sprint Reports

This directory contains engineering implementation reports for every ERP sprint.

Release Notes remain inside

`docs/07_RELEASES/`

Sprint Reports are archived separately for engineering history.

## Folder structure

```text
Sprint_25/
Sprint_26/
Sprint_27/
...
```

## Each sprint folder contains

- README
- Phase Completion Reports
- Validation Report
- Validation Fix Report

Beginning with Sprint 25 this becomes the official documentation standard.
